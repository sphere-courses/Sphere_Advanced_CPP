# Ping-Pong
Данная программа - пример работы многонитевой программы.
### Компиляция:
Для выполнения компиляции достаточно выполнить следующую команду:
```
IN: ~/make
```
### Пример использования:
```
IN: ~/ping_pong
OUT: ping pong ping pong ping pong ...
```
```
IN: ~/ping_pong 3
OUT: ping pong ping
```
